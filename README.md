## 🤖 About This Bot
Milon Bot একটি powerful Messenger automation bot, যেটা বিভিন্ন smart command, AI system, song search এবং fun feature support করে।

---

## ⚙️ Features
- 🤖 AI Chat System  
- 🎵 Song Search & Play  
- 📊 Auto Info & Activity  
- ⚡ Fast Response  
- 🛠️ Easy Custom Command  

---

## 🚀 Installation

```bash
git clone https://github.com/milon220/Ghost.git
cd Ghost
npm install
npm start
```

---

## 📌 Command Example

```
!help
!ai hello
!song faded
```
